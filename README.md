# Data_Scraper_for_GS
# Google Scholar Scraper

A modular tool for extracting publication metadata from Google Scholar.  
This repository provides documentation and setup instructions so you can quickly build and extend your own Scholar scraper.

---

## Features
- Fetch author profiles, publication titles, citations, and links
- Configurable search queries (keywords, author names, institutions)
- Export results to CSV/JSON for downstream analysis
- Modular design for easy extension
